# File Permissions
___
Class: [[OS]]
Type: 
Tags: # 
Date: March 6th, 2025
___

Entities: user, group, other
Each entity has the following permissions: *r*(ead), *w*(rite), (e)*x*(ecute)
For each entity, the permissions can be specified in octal 

Example: chmod 700 file.txt changes the permissions to:
user: rwx
group: ---
other: ---
